Design System

A complete visual language for Wayward — an LLM-powered tabletop narrator interface. All colors, typography, spacing, and component patterns documented for implementation in any environment.

01 Philosophy
Manuscript at Night
Dark, warm, gold-accented. Evokes old parchment and candlelight without being literal about it.
Amber Warmth
No gradients on backgrounds, no neon, no hard whites. Every surface carries amber warmth.
Typographic Hierarchy
Three distinct typefaces: display, body, and UI chrome — each with a precise role, never mixed.
Core Principle
Every surface has amber warmth. Text is never pure white. Backgrounds are never pure black. The entire palette exists on a narrow warm-amber axis — even the coolest accent (steel blue, --blue) leans warm.

Chat · fluid
