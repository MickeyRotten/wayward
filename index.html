  1. INSTALL DEPENDENCIES

    Run once after cloning the repo.

    pip install -r server/requirements.txt
cd client && npm install

  2. START THE BACKEND

    Starts the FastAPI server on port 8000. Creates the database and seeds default content on first run.

    python -m uvicorn server.main:app --reload

  3. START THE FRONTEND

    Starts the Vite dev server on port 5173. API calls proxy to the backend automatically.

    cd client && npm run dev

  4. CONFIGURE & PLAY

    Open the app, click Settings, enter your OpenRouter API key, pick a model, and start your adventure.
